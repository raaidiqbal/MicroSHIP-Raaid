Make sure you put the right name of the Anime in the search bar!

For tech-savvier people, remember that the URL convention for basic features is /title?title=(name of title) or /genres?genre=(genres to search by)

For example, /title?title=Trigun returns information of an Anime with the title Trigun
/genres?genre=Action returns a table of Anime titles whose genre is Action
/genres?genre=Action&genre=Comedy returns a table of Anime titles whose genre is action AND comedy!
